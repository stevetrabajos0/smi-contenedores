- **Resolución recomendada**: 1200x900px (aspect ratio 4:3)
- **Peso máximo**: 200KB por imagen (optimizar con TinyPNG o similar)
- **Formato**: JPG optimizado

## Instrucciones para el usuario

1. **Temporal**: Descarga imágenes placeholder de Unsplash/Pexels basadas en las descripciones
2. **Producción**: Reemplaza con fotos reales de proyectos SMI cuando estén disponibles
3. **Optimización**: Usa [TinyPNG](https://tinypng.com) para comprimir antes de subir
4. **Nombres**: Mantén exactamente los nombres listados arriba (sin espacios, minúsculas)
